unparse :: Bool -> Bool -> Program -> TextSource

unparse fully-paren indent program synthesizes the MOO code corresponding to the given abstract syntax tree. If fully-paren is true, all expressions are fully parenthesized, even when unnecessary given the circumstances of precedence. If indent is true, the resulting MOO code will be indented with spaces as appropriate to show the nesting structure of statements.

The MOO code is returned as a single Text value containing embedded newline characters.
